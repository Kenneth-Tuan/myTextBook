# Verse

    V1 正當那時，有人將彼拉多使加利利人的血攙雜在他們祭物中的事告訴耶穌。
    V2 耶穌說；你們以為這些加利利人比眾加利利人更有罪，所以受這害嗎？
    V3 我告訴你們，不是的！你們若不悔改，都要如此滅亡！
    V4 從前西羅亞樓倒塌了，壓死十八個人；你們以為那些人比一切住在耶路撒冷的人更有罪嗎？
    V5 我告訴你們，不是的！你們若不悔改，都要如此滅亡！
    V6 於是用比喻說：一個人有一棵無花果樹栽在葡萄園裡。他來到樹前找果子，卻找不著。
    V7 就對管園的說：看哪，我這三年來到這無花果樹前找果子，竟找不著。把他砍了吧，何必白佔地土呢！
    V8 管園的說：主啊，今年且留著，等我周圍掘開土，加上糞；
    V9 以後若結果子便罷，不然再把他砍了。

當我們在新聞上看到天災人禍、意外事故，或是身邊有人突然遭遇不幸時，我們內心第一個反應往往是：「為什麼會這樣？」在猶太人的傳統觀念中，有一種根深蒂固的「因果報應」思想——如果一個人遭遇極大的不幸，那他一定犯了什麼特別嚴重的罪。
V1「正當那時」：就是正當主耶穌勸人要盡力與對頭了結之時(參十二58~59)。
「彼拉多使加利利人的血攙雜在他們祭物中的事」：可能在彼拉多任猶太巡撫時，曾經為了鎮壓加利利人的暴亂，在他們獻祭時出動軍兵屠殺，以致被殺者的血攙雜在他們所獻的祭物中。
這是一起極其殘忍的政治與宗教悲劇。人們帶著這個話題來找耶穌，背後其實隱含的意思是：「耶穌，這些加利利人是不是做了什麼壞事，才會落得如此下場？」
耶穌看穿了他們的想法，並藉此帶出一個翻轉人心的信息：苦難不是用來衡量別人罪惡的尺，而是上帝的敲打，讓我們悔改的警鐘
耶穌直接指出了核心問題：
「你們以為這些加利利人比眾加利利人更有罪，所以受這害嗎？我告訴你們：不是的！你們若不悔改，都要如此滅亡！」（路13:2-3）
為了加深印象，耶穌自己舉了另一個例子：西羅亞樓倒塌，壓死了十八個人。這不是人為蓄意的屠殺，而是一場突如其來的「天災意外」。耶穌再次強調：這些人並不比住在耶路撒冷的一切人更有罪。我告訴你們，不是的！你們若不悔改，都要如此滅亡！
意外的發生不是要我們去評斷死者的罪，而是要提醒生者——生命無常，我們是否預備好面對審判？若不悔改，人人都有滅亡的結局
接著，耶穌講了一個「無花果樹」的比喻。一個人把無花果樹栽在葡萄園裡，他三年來都去樹上找果子，卻什麼也找不到。
園主的反應： 「把它砍了吧，何必白佔地土呢？」不結果子的樹，就失去了存在的價值。
管園的求情： 「主啊，今年且留著，等我周圍掘開土，加上糞；以後若結果子便罷，不然再把它砍了。」（路13:8-9）
我們就是那棵不結果子的無花果樹。我們信主多年，或者享受了上帝極多的恩典，生命卻可能充滿了自私、冷漠與苦毒，沒有結出與悔改相稱的果子。本該砍掉的，但那為我們求情的「管園者」就是耶穌基督，向父神呼求：「再給他一年的機會！」上帝的恩典不是放任我們腐爛，而是「掘開土，加上糞」——有時候，我們生命中遇到的管教、挫折、或是神話語的扎心（掘土施肥），雖然痛苦，卻是主耶穌在向我們施恩，渴望我們重新活過來的恩典。
悔改是「心意轉變」、「調轉方向」。它不只是在情感上覺得虧欠、流淚禱告而已，更是意志上的轉向。
從自私轉向給予： 過去只顧自己的利益，現在開始看見別人的需要。
從苦毒轉向饒恕： 抓著別人的過錯不放，現在靠著主選擇放下與祝福。
從慾望的追求轉向對上帝的敬畏： 每天的生活不再是被欲望拽著走，而是誠誠實實地面對神。
弟兄姊妹，今天我們還能坐在這裡敬拜、讀經，不是因為我們比在新聞裡看到遭遇不幸的人更優秀，而是管園的主為我們爭取了「再留一年」的恩典。
不要把上帝的寬容當作理所當然，也不要總是在評斷別人的罪。讓我們把目光收回到自己的生命中，對主說：「主啊，謝謝祢還留著我。求祢鬆開我剛硬的心土，施肥餵養我，讓我的生命在今年，為祢結出悔改與仁愛的果子。

# 禱讀經文

還是你藐視他豐富的恩慈、寬容、忍耐，不曉得他的恩慈是領你悔改呢？ - 羅馬書 2:4

# 額外的解釋

    在路加福音第13章1-9節那邊，為什麼前面說要這些加利利人悔改，後面接了一個無花果樹的例子？其中的關聯性為何？

先看 1-5 節的論證核心
當有人報告比拉多殺加利利人的事，背後帶著一個隱含的神學假設：這些人遭難，是因為他們罪孽更深。這幾乎是當時猶太人普遍的因果神學框架（約伯記的朋友們，或約 9:2 問耶穌的門徒，都是同樣邏輯）。
耶穌連著用兩個例子——被殺的加利利人、西羅亞樓塌壓死的十八個人——兩次給出相同的答案：

「你們若不悔改，都要如此滅亡。」

關鍵在於**「如此」（ὁμοίως, homoiōs）**這個詞。耶穌並不是說「你們也會被羅馬兵殺」或「也會被樓壓死」，而是說：突然而來、毫無準備的滅亡，就是你們的命運——除非你們悔改。
他等於做了一個修辭上的反轉：「問題不是他們比你們壞，問題是你們跟他們一樣，都站在審判的邊緣。」
這就自然引出一個張力：上帝目前為何還沒有動作？為什麼還有機會悔改？

無花果樹的比喻正是在回答這個問題
無花果樹在舊約裡是以色列的象徵這點廣為人知（何 9:10、彌 7:1、耶 8:13），路加的讀者對這個意象不會陌生。比喻的結構是：

園主（上帝）三年尋果不得，要求砍樹
園丁（多數學者認為是耶穌作為代求者的形象）請求再給一年，施肥挖土，再看能否結果
最終裁決留待那「一年」之後

這個比喻與前段的邏輯扣合非常緊密：
前段問：為什麼那些人死了，你們還活著？
比喻答：不是因為你們更好，而是因為還在那「多給的一年」之中。
前段說：若不悔改，必要滅亡。
比喻說：滅亡不是上帝的第一意願，但也絕非空話——園主的命令清清楚楚，是園丁在為樹求情。
前段的對象是「你們眾人」。
比喻把論點從個人事件拉升到整體以色列的歷史處境：這棵樹已經佔地三年了（許多詮釋者將「三年」對應耶穌事工的年限，儘管這不能確定），一直沒有結果。現在給的這一段時間，是恩典的空間，不是理所當然的。

兩段之間真正的連結邏輯
可以這樣來理解這兩段話的整體推進：
第一段（1-5節）拆解了「災難＝罪的懲罰」這個錯誤框架，把問題倒回給所有聽眾：你們自己呢？這是緊迫性的建立。
第二段（6-9節）解釋了「那為什麼還有機會」這個問題：因為有代求，因為上帝給了限期的恩典空間。但這個空間不是無限的。這是機遇性與有限性的揭示。
兩段合起來的信息結構是：

> 你們不要以為自己比那些遭難的人更有理由活著。你們活著，是因為上帝在等待悔改的果子。但這個等待是有限期的——園丁正在代求，但園主的斧頭仍在那裡。

# 我的領受

1. 我擁有豐盛的生命，我的工作、家庭、服事、身體各方面都充滿神的祝福和恩典，所以我想要更多的幫助那些有需要的人。
2. 很具體的事件：對於喜樂小組、敬拜團的情況，跟人的相處中，我的心、和動機是真的純正嗎？是否是自私的、苦毒、咒詛或論斷？
3. 求主耶穌今天再一次的用祢的寶血潔淨我，我的私慾、驕傲、小心、論斷、想要掌控、不合一這類的罪都要在祢面前陳明，因祢的大能和恩典，這些罪今天都要從我的身上再一次的除去。
4. 我需要有更多的憐憫，主耶穌基督的憐憫，使我看到的不只是人的罪、自己的罪，而是看見在罪的背後那更深處的需要。
5. 疾病、災難是會臨到的，無人可避免，且這也是在世上神所允許的，我們還沒有經歷，是因為有耶穌的代求、有了有限的恩典空間。
