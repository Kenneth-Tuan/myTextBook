- **主題（Topic）**：
    - 整個文檔的主要主題或標題。

- **引言（Introduction）**：
    - 介紹背景、問題、目標或主題的重要性。

- **主要部分（Main Body）**：
    - **小標題或章節**：針對不同的要點進行詳細描述，通常會根據話題進行分段。
    - **子標題**：對應主要部分的細分，提供更具體的詳細說明。

- **結論（Conclusion）**：
    - 總結主要觀點、提供結論或解決方案，並可能展望未來的研究或討論方向。

- **附錄（Appendix，若有需要）**：
    - 提供額外的支持材料、數據、圖表或文獻。

- **參考文獻（References）**：
    - 引用的書籍、文章或資料來源。
