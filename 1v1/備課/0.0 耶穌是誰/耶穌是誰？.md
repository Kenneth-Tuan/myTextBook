# 描寫耶穌的經文
他禁食四十晝夜，後來就餓了。（馬太福音4:2）
在那裏有雅各井；耶穌因走路困乏。就坐在井旁。那是約有午正。
耶穌在船尾上，枕著枕頭睡覺。門徒叫醒了他，說：「父子！我們喪命，你不顧嗎？」（馬可福音4:38）
耶穌哭了（約翰福音11:35）

# 耶穌如何描述自己？
>1. 我與父原為一（約翰福音10:30）
>2. 叫人都尊敬子如同尊敬父一樣。不尊敬子的，就是不尊敬拆子來的父。（約翰福音5:23）
>3. 耶穌對他說：「菲力，我與你們同在這樣長久，你還不認識我嗎？人看見了我，就是看見了父；你怎麼說『將父顯給我們看』呢？

# 耶穌的同伴認為祂是誰呢？
> 1. 多馬說：「我的主！我的神！」（約翰福音20:28）
> 2. 百夫長和一同看守耶穌的人看見地震並所經歷的事，就極其害怕，說：「這真是神的兒子了！」（馬太福音27:54）
> 3. 在船上的人都拜他，說：「你真是神的兒子了。」（馬太福音14:33）
###### 歸納一下：
認為耶穌是神的兒子。
